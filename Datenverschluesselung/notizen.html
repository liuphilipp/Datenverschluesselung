Caesar-Verschlüsselung: Jeder Buchstabe wird um einen festen Wert verschoben. Aus A wird z.B. D (bei Verschiebung um 3). Das Passwort bestimmt die Verschiebung (= Länge des Passworts in Zeichen).
Reverse: Der Text wird einfach rückwärts geschrieben. Aus Hallo wird ollaH. Sehr einfach, aber gut als Einstieg.
Base64: Der Text wird in ein Format aus Buchstaben, Zahlen und Sonderzeichen umgewandelt. Aus Hallo wird SGFsbG8=. Wird häufig zur sicheren Datenübertragung genutzt.
ROT13: Eine Caesar-Variante — jeder Buchstabe wird genau um 13 Stellen verschoben. Besonderheit: Nochmals ROT13 → wieder der Originaltext! Aus Hallo wird Unyyb.
Morse-Code: Jeder Buchstabe wird durch Punkte (·) und Striche (−) dargestellt. Ursprünglich für Telegrafie. Aus SOS wird ··· −−− ···.
Binär-Code: Jeder Buchstabe wird in seine 8-stellige Binärzahl übersetzt. So arbeitet ein Computer intern. Aus A wird 01000001.
